# Privacy Policy

**Research Reviewer** is a Claude Code skill consisting solely of local instruction files
(Markdown). It does not collect, store, transmit, or process any personal data. It has no
backend, no servers, no analytics, and no telemetry. It makes no network requests of its own.

All review activity happens locally within your Claude Code session and is governed by
[Anthropic's Privacy Policy](https://www.anthropic.com/legal/privacy). The author of this
plugin receives no data from your use of it.
